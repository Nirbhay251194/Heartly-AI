# PART 5 — UI/UX DESIGN SYSTEM & DESIGN LANGUAGE

---

# SECTION 55 — DESIGN PHILOSOPHY

Hartly AI is not a chatbot.

Hartly AI is an emotional companion.

The user should feel that they are opening a premium messaging application instead of an AI product.

Every design decision should reinforce emotional comfort, simplicity, warmth and trust.

The interface should feel:

• Modern

• Premium

• Calm

• Romantic

• Human

• Minimal

Never make the application feel like a dashboard.

Never overload the user with options.

Conversation should always remain the primary focus.

---

# SECTION 56 — DESIGN PRINCIPLES

The UI should follow these principles.

1. Conversation First

The chat interface must always receive the highest visual priority.

2. Less is More

Avoid unnecessary buttons.

Avoid unnecessary menus.

3. Mobile First

Every screen should be designed for mobile before desktop.

4. Emotional Experience

Spacing, typography and colors should reduce anxiety.

5. Familiar Interface

Users should immediately understand the interface without learning.

WhatsApp and Telegram are strong inspirations.
